## 🚀 실행 방법

### 1. 패키지 설치

```bash
cd admin-web
npm install
```

### 2. 환경 변수 설정

`.env` 파일 생성:
```bash
cp .env.example .env
```

`.env` 파일 내용:
```
REACT_APP_API_URL=http://localhost:8000/api
```

### 3. 개발 서버 실행

```bash
npm start
```

개발 서버가 http://localhost:3000 에서 실행됩니다.

## 🔗 백엔드와 연결

백엔드 서버가 http://localhost:8000 에서 실행 중이어야 합니다.

백엔드 서버 실행:
```bash
cd backend
source venv/bin/activate
python run.py
```

## 📊 주요 기능

### 인증 시스템
- JWT 기반 인증
- 로그인/로그아웃
- 자동 로그인 유지
- 권한별 메뉴 표시

### 권한 관리
- **슈퍼관리자**: 모든 기능 접근 가능
- **일반 관리자**: 관리 기능 접근 가능
- **영업자/기사**: 제한된 기능만 접근

### CRUD 기능
- 모든 관리 항목에 대해:

  - 상세 조회

  - 삭제

### 데이터 연관
