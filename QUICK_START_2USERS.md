# 2명이 함께 사용하기 - 빠른 시작 가이드

## 🚀 간단한 설정 (5분 안에 완료)

### 1단계: 서버 컴퓨터 설정

**서버 컴퓨터 (메인 컴퓨터)에서:**

```bash
# 터미널 1: 백엔드 서버 실행
cd backend
source venv/bin/activate
python run.py
```

```bash
# 터미널 2: 관리자 웹 실행
cd admin-web
npm start
```

### 2단계: 서버 컴퓨터 IP 주소 확인

**서버 컴퓨터에서:**

- **macOS/Linux:**
  ```bash
  ifconfig | grep "inet " | grep -v 127.0.0.1
  ```
  예: `inet 192.168.0.100`

- **Windows:**
  ```bash
  ipconfig
  ```
  "IPv4 주소" 찾기, 예: `192.168.0.100`

**이 IP 주소를 기록해두세요!** (예: `192.168.0.100`)

### 3단계: 다른 컴퓨터에서 접속

**다른 컴퓨터의 브라우저에서:**

```
http://서버IP주소:3000
```

예: `http://192.168.0.100:3000`

### 4단계: 로그인

- **기본 관리자 계정:**
  - 사용자명: `admin`
  - 비밀번호: `admin123`

## ✅ 완료!

이제 두 컴퓨터에서 동시에 사용할 수 있습니다!

## 📝 주의사항

1. **서버 컴퓨터는 항상 켜두세요**
   - 서버 컴퓨터를 끄면 다른 컴퓨터에서 접속할 수 없습니다.

2. **같은 네트워크에 연결되어 있어야 합니다**
   - 같은 Wi-Fi 또는 이더넷 네트워크에 연결되어 있어야 합니다.

3. **방화벽 설정**
   - 서버 컴퓨터의 방화벽에서 포트 8000, 3000을 허용해야 할 수 있습니다.

## 🔧 문제 해결

### 접속이 안 될 때:

1. **서버 컴퓨터에서 서버가 실행 중인지 확인**
   - 백엔드: `http://localhost:8000/docs` 접속 확인
   - 관리자 웹: `http://localhost:3000` 접속 확인

2. **IP 주소 확인**
   - 서버 컴퓨터에서 `ifconfig` (macOS/Linux) 또는 `ipconfig` (Windows) 실행

3. **네트워크 확인**
   - 같은 네트워크에 연결되어 있는지 확인
   - `ping 서버IP주소` 명령어로 연결 확인

### CORS 오류가 발생할 때:

- 백엔드 서버를 재시작하세요.

## 💡 팁

- 서버 컴퓨터의 IP 주소가 변경되지 않도록 네트워크 설정에서 고정 IP를 사용하는 것을 추천합니다.
- 더 자세한 설정은 `NETWORK_SETUP_GUIDE.md`를 참고하세요.
