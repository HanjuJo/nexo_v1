# 넥소코리아 고객관리 관리자 웹

React + TypeScript 기반 관리자 웹 애플리케이션

## 설치

```bash
npm install
```

## 실행

```bash
npm start
```

개발 서버가 http://localhost:3000 에서 실행됩니다.

## 환경 변수

`.env` 파일을 생성하여 다음 변수를 설정하세요:

```
REACT_APP_API_URL=http://localhost:8000/api
```

## 주요 기능

### 관리자 웹 (20개 모듈)

1. **관리자 로그인** - 로그인 및 자동 로그인
2. **관리자 계정 관리 (슈퍼관리자)** - 관리자 계정 목록 조회 및 등록 버튼
3. **관리자 계정 관리 상세 (슈퍼관리자)** - 관리자 계정 등록, 수정, 삭제
4. **직원 관리 리스트** - 직원 목록, 이름으로 검색, 등록 버튼
5. **직원 관리 상세** - 직원 정보 조회, 등록, 수정, 삭제
6. **거래처 관리 리스트** - 거래처 목록, 이름으로 검색, 등록 버튼
7. **거래처 관리 상세** - 거래처 정보 조회, 등록, 수정, 삭제 (개인/기업/기관 구분)
8. **품목 관리 리스트** - 품목 목록, 등록 버튼
9. **품목 관리 상세** - 품목 정보 조회, 등록, 수정, 삭제
10. **상담 관리 리스트** - 상담 목록, 직원/거래처명으로 검색, 등록 버튼
11. **상담 관리 상세** - 상담 정보 조회, 등록, 수정, 삭제
12. **견적 관리 리스트** - 견적 목록, 직원/거래처명으로 검색, 등록 버튼
13. **견적 관리 상세** - 견적 정보 조회, 등록, 수정, 삭제
14. **계약 관리 리스트** - 계약 목록, 직원/거래처명으로 검색, 등록 버튼
15. **계약 관리 상세** - 계약 정보 조회, 등록, 수정, 삭제
16. **설치 및 AS 관리 리스트** - 설치/AS 목록, 직원/거래처/설치유형/AS유형으로 검색
17. **설치 및 AS 관리 상세** - 설치/AS 정보 조회, 등록, 수정, 삭제
18. **재고 관리 리스트** - 재고 목록, 재고 등록 버튼
19. **재고 관리 상세** - 재고 수량 수동 등록, 수정, 삭제
20. **로그아웃** - 로그아웃 기능

## 프로젝트 구조

```
admin-web/
├── src/
│   ├── components/     # 공통 컴포넌트
│   │   ├── Layout.tsx  # 레이아웃 (사이드바, 헤더)
│   │   └── common/     # 공통 UI 컴포넌트
│   ├── contexts/       # React Context
│   │   └── AuthContext.tsx  # 인증 컨텍스트
│   ├── pages/          # 페이지 컴포넌트
│   │   ├── Login.tsx   # 로그인 페이지
│   │   ├── Dashboard.tsx  # 대시보드
│   │   ├── admin/      # 관리자 계정 관리
│   │   ├── employees/  # 직원 관리
│   │   ├── clients/    # 거래처 관리
│   │   ├── items/      # 품목 관리
│   │   ├── consultations/  # 상담 관리
│   │   ├── quotations/     # 견적 관리
│   │   ├── contracts/      # 계약 관리
│   │   ├── installations/  # 설치/AS 관리
│   │   └── inventory/      # 재고 관리
│   └── services/       # API 서비스
│       └── api.ts      # Axios 설정
├── public/
└── package.json
```

## 기술 스택

- React 18
- TypeScript
- React Router v6
- Axios
- CSS3
